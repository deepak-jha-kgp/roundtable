## 🚀 Install

**One click** — press **Install to Lemma** above (or [open the installer](https://lemma.work/import/github/deepak-jha-kgp/roundtable)).

**From inside Lemma** — go to **Settings → Share & Export → Import from GitHub** and paste `deepak-jha-kgp/roundtable`.

Everything installs into *your own* workspace — your data, your model keys. Connectors reconnect to *your* accounts and apps rebuild for your pod on import, so nothing is shared but the recipe.
